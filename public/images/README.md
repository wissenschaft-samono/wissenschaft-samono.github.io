# 画像置き場

ヘッダー背景写真は、このフォルダに次の名前で置いてください。

```text
header-background.jpg
```

記事で写真を使う場合も、このフォルダに画像を置きます。

例:

```text
public/images/bath-note-01.jpg
```

記事本文ではこう書きます。

```markdown
![写真の説明](/images/bath-note-01.jpg)
```

注意:

- 自分で撮った写真、または掲載してよい写真だけを置く。
- 顔、住所、個人情報が写っていないか確認する。
- 書籍ページ、論文PDF、書影、他人の写真を無断で置かない。
